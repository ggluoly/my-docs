# XXL-Job / 定时任务

## 是什么

定时任务负责按计划周期性执行业务逻辑。分布式项目中通常需要分布式任务调度平台，支持任务管理、分片执行和失败重试。

## 常见组件

- `XXL-Job`：分布式任务调度。
- `Quartz`：Java 定时任务框架。
- `Spring Scheduler`：Spring 自带简单定时任务。
- `ElasticJob`：分布式任务调度。
- `PowerJob`：分布式任务调度。

## 典型场景

```
订单超时取消
账单生成
定时报表
数据同步
定时清理缓存
定时推送消息
每日统计
```

## 选型建议

简单项目可以用：

```
Spring Scheduler
```

中大型项目推荐：

```
XXL-Job
```

XXL-Job 提供可视化调度后台、任务分片、故障转移和执行日志，比 `@Scheduled` 更适合多实例部署的微服务环境。
