# MinIO / 对象存储

## 是什么

对象存储服务，负责图片、附件、合同、Excel、PDF 等文件的存储。企业项目很容易遗漏文件存储，但实际非常常用。

## 常见组件

- `MinIO`：私有化对象存储。
- `阿里云 OSS`：云对象存储。
- `腾讯云 COS`：云对象存储。
- `AWS S3`：云对象存储。
- `FastDFS`：老项目常见文件系统。

## 典型场景

```
头像上传
合同文件
Excel 导入导出
PDF 文件
图片资源
视频文件
附件管理
```

## 选型建议

```
私有化部署：MinIO
云上部署：OSS / COS / S3
```

MinIO 兼容 S3 协议，私有化部署时可以用同一套 S3 客户端代码，后期迁移到云对象存储成本较低。
