# 分库分表

## 是什么

当单表数据量过大时，可能需要分库分表来分散存储和查询压力。

## 同类组件

| 组件 | 说明 |
| --- | --- |
| `Apache ShardingSphere` | 分库分表、读写分离、数据加密，新项目常见选择 |
| `MyCat` | 分库分表中间件，老项目常见 |
| `Vitess` | MySQL 分片方案 |

## 典型场景

```
订单表
交易流水表
用户行为表
日志表
消息表
```

## 选型建议

新项目常见选择：

```
Apache ShardingSphere
```

但不要过早使用分库分表，只有数据规模确实需要时再引入，否则只会徒增复杂度。
