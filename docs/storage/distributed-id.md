# 分布式 ID

## 是什么

企业微服务项目通常需要全局唯一 ID，单库自增主键无法满足跨库、跨服务的唯一性要求。

## 同类组件

| 组件 | 说明 |
| --- | --- |
| `Snowflake` | 雪花算法 ID |
| `Leaf` | 美团分布式 ID |
| `Redis INCR` | Redis 自增 ID |
| `数据库号段模式` | 批量获取 ID 段 |
| `UUID` | 通用唯一 ID |

## 常见场景

```
订单号
支付流水号
用户编号
业务单号
分库分表主键
```

## 选型建议

常见选择：

```
雪花算法
Leaf
号段模式
```

不建议所有业务都直接用 UUID，因为排序性和索引效率不一定好。
