## 收录原则

- 优先收录官方文档、开源项目主页和长期稳定的技术教程。
- 不收录短期活动页、广告页、下载站镜像或来源不明的资料。
- 链接失效、内容长期不维护或明显偏离 Java 后端技术栈时，应及时移除。
