# RocketMQ

## 是什么

RocketMQ 是阿里开源的分布式消息中间件，支持业务消息、事务消息、延迟消息，是国内企业和电商场景最常用的消息队列之一。

## 解决什么问题

消息队列主要解决这些问题：

```
异步解耦
削峰填谷
订单超时
库存扣减
消息通知
日志采集
事件驱动
最终一致性
```

## 定位差异

RocketMQ 更偏订单、交易和最终一致性等业务消息，Kafka 更偏高吞吐日志与事件流，Pulsar 更偏多租户和计算存储分离的平台型架构。RabbitMQ 仅作为传统 AMQP 方案参考与面试考点。完整对比见 [消息与事务：MQ 对比与选型](/messaging/#mq-对比与选型)。

## 选型建议

国内企业和电商业务消息场景，RocketMQ 是常见首选，尤其是需要事务消息、延迟消息时。
