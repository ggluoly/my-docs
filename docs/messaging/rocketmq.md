# RocketMQ

## 是什么

RocketMQ 是阿里开源的分布式消息中间件，支持业务消息、事务消息、延迟消息，是国内企业和电商场景最常用的消息队列之一。

## 解决什么问题

消息队列主要解决这些问题：

```
异步解耦
削峰填谷
订单超时
库存扣减
消息通知
日志采集
事件驱动
最终一致性
```

## 同类组件对比

| 组件 | 定位 | 适用场景 |
| --- | --- | --- |
| `RocketMQ` | 业务消息、事务消息、延迟消息 | 国内企业和电商常用 |
| `Kafka` | 高吞吐日志、埋点、流式数据 | 大数据、日志、实时计算，详见 [Kafka](./kafka) |
| `Pulsar` | 云原生消息与流式数据平台 | 多租户、消息与流统一、分层存储，详见 [Apache Pulsar](./pulsar) |
| `RabbitMQ` | 普通异步任务、通知、延迟队列 | 中小系统常用 |
| `Spring Cloud Stream` | 消息中间件抽象 | 多 MQ 统一接入 |

## 选型建议

```
业务系统：RocketMQ / RabbitMQ
日志和大数据：Kafka
云原生多租户和消息流统一：Pulsar
```

国内企业和电商业务消息场景，RocketMQ 是常见首选，尤其是需要事务消息、延迟消息时。
