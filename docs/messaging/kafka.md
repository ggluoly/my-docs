# Kafka

## 是什么

Kafka 是高吞吐的分布式消息系统，擅长处理日志、埋点、流式数据，适合大数据、日志采集、实时计算场景。

## 解决什么问题

和其他消息队列一样，Kafka 用于异步解耦、削峰填谷、事件驱动等，但它的强项在高吞吐和流式数据处理。

## 同类组件对比

| 组件 | 定位 | 适用场景 |
| --- | --- | --- |
| `Kafka` | 高吞吐日志、埋点、流式数据 | 大数据、日志、实时计算 |
| `RocketMQ` | 业务消息、事务消息、延迟消息 | 国内企业和电商，详见 [RocketMQ](./rocketmq) |
| `RabbitMQ` | 普通异步任务、通知、延迟队列 | 中小系统常用 |
| `Spring Cloud Stream` | 消息中间件抽象 | 多 MQ 统一接入 |

## 选型建议

```
业务系统：RocketMQ / RabbitMQ
日志和大数据：Kafka
```

当系统涉及大量日志采集、用户行为埋点、实时流式计算时，Kafka 是常见选择。它也常配合实时计算引擎（如 Flink、Kafka Streams）使用，详见[数据分析](../deploy/data-analysis)相关章节。
